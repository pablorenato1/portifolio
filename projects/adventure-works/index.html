Full Pipeline

Pipeline de dados end-to-end usando a base de dados Adventure Works — da extração via Meltano, orquestração com Airflow, transformação com dbt, armazenamento no Databricks, até o dashboard final no Power BI.

// 02 DAGs do Airflow

DAG Descrição Schedule
meltano_extraction_separated_tasks_dag Extrai dados da API e do MSSQL via Meltano e carrega na camada Raw do Databricks @daily
dbt_build_dag Executa os modelos de transformação dbt e aplica testes de qualidade de dados @daily

// 03 Camadas no Databricks

Raw Dados brutos ingeridos diretamente pelo Meltano, sem transformações.
Staging Dados limpos, tipados e padronizados. Transformações básicas aplicadas via dbt.
Intermediate Aplicação de regras de negócios mais robustas nos dados.
Marts Modelos analíticos prontos para consumo — fatos e dimensões otimizados para Power BI.

// 04 Modelos dbt (Gold)

fact_sales_orders

Fato

Modelo central de pedidos de venda. Integra dimensões de cliente, produto, endereço, promoções e taxas de câmbio para análise transacional completa.

fact_promotions_usage

Fato

Rastreia a efetividade de campanhas promocionais, conectando ofertas, produtos, clientes e tempo para insights de performance.

// 05 Dashboard Power BI

Adventure Works — Dashboard de Vendas

Dashboard analítico conectado ao Databricks Gold layer, com métricas de vendas, desempenho de produtos e análise de campanhas.
