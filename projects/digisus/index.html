// 02 Objetivos do Projeto

01. Consumir a API do Digisus filtrando exclusivamente os dados do estado de Pernambuco, evitando o volume desnecessário de dados nacionais.
02. Estruturar um pipeline de tratamento que padronize, limpe e organize os dados de saúde para análise.
03. Armazenar os dados processados no Databricks com estrutura organizada para consultas analíticas.
04. Construir um dashboard em Power BI que permita visualizar indicadores de saúde de Pernambuco de forma clara e acessível.

// 04 Sobre o Digisus

O Digisus é o sistema de gestão do SUS (Sistema Único de Saúde) no Brasil, responsável por centralizar dados de saúde pública municipal e estadual, incluindo informações sobre atenção básica, programas de saúde e indicadores de cobertura.

Este projeto foca exclusivamente nos dados de Pernambuco, permitindo uma análise mais detalhada e contextualizada da situação da saúde pública no estado.
